# Admin Panel

Path: `admin_panel/`

## Stack
- React + Vite + TypeScript
- Firebase Firestore metadata writes
- Transcript parse and timing workflow

## Key Features Implemented
- Upload local MP3 for timing session.
- Paste LRC and parse to timed segments.
- Keyboard shortcuts:
  - `Space` play/pause
  - `Enter` mark timing and move next
  - `Backspace` previous line
  - `ArrowLeft/Right` seek
  - `Shift + Arrow` larger seek
  - `Ctrl + Arrow` fine seek
  - `F` flag line
- JSON export/copy for transcript publishing.
