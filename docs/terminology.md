# Nitnem Terminology

This document defines key terms used across the Nitnem project to ensure consistent communication.

- **Nitnem:** The set of daily prayers performed by Sikhs.
- **Transcript:** The text content of a prayer, often synchronized with audio.
- **Sync Engine:** The core component responsible for downloading and updating content from Firebase to local storage.
- **Content Item:** A unit of spiritual content (e.g., a specific prayer) managed in the catalog.
- **Track:** A specific audio/transcript version of a prayer.
- **Local-First:** An architectural approach where data is available offline and synchronized in the background.
