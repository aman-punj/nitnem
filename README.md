# Nitnem

Nitnem is a content-driven Sikh prayer application with a React-based administration dashboard.

## Repository Structure

- `mobile_app/`: Flutter mobile application.
- `admin_panel/`: React/Vite administration dashboard.
- `shared/`: Shared schemas and transcript samples.
- `docs/`: Technical documentation.

## Getting Started

### Mobile App
1. `cd mobile_app`
2. `flutter pub get`
3. `flutter run`

### Admin Panel
1. `cd admin_panel`
2. `npm install`
3. Copy `.env.example` to `.env` and fill values.
4. `npm run dev`

## Documentation
See the `docs/` directory for detailed documentation on architecture, sync engine, and more.
